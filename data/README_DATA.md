# Dataset Information

## Source

Les données utilisées dans ce projet proviennent du dataset :

**E-commerce Behavior Data from Multi Category Store**

## Description

Le dataset contient des événements utilisateurs enregistrés sur une plateforme e-commerce.

Chaque ligne correspond à une action réalisée par un utilisateur :

* view : consultation d'un produit
* cart : ajout d'un produit au panier
* purchase : achat d'un produit

## Période couverte

Octobre 2019 à Avril 2020

## Données utilisées

Pour des raisons de mémoire et de performance, un échantillon de 100 000 lignes a été utilisé pour chacun des 7 fichiers mensuels.

## Disponibilité

Les fichiers CSV ne sont pas inclus dans ce dépôt GitHub en raison de leur taille.

Ils peuvent être téléchargés depuis la source d'origine du dataset.
